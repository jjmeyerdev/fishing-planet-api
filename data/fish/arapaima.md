# Arapaima

## How to catch

How to catch **a unique Arapaima**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Arapaima would be to use **Large Minnows** on a size #10/0 hook.

If you are more into artificial lures, you should go for some **Jigging Spoons**. The most successful lure is **Blue/Silver Jigging Spoon 71 g, #7/0**.
