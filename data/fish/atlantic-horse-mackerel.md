# Atlantic Horse Mackerel

## How to catch

How to catch **a unique Atlantic Horse Mackerel**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Atlantic Horse Mackerel would be to use **Sea Worms** on a size #2/0 hook.

If you are more into artificial lures, you should go for some **Worms**. The most successful lure is **Dim Green / Glow Green Worm 5 cm**.
