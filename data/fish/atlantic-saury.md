# Atlantic Saury

## How to catch

How to catch **a unique Atlantic Saury**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Atlantic Saury would be to use **Ragworms** on a size #1 hook.

If you are more into artificial lures, you should go for some **Worms**. The most successful lure is **Dim Green / Glow Green Worm 5 cm**.
