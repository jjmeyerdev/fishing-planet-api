# Bonefish

## How to catch

How to catch **a unique Bonefish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Bonefish would be to use **Scaled Sardines** on a size #6/0 hook.

If you are more into artificial lures, you should go for some **Buzzbaits**. The most successful lure is **White Buzzbait 28 g, #5/0**.
