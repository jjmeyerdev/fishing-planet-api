# Bowfin

## How to catch

How to catch **a unique Bowfin**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Bowfin would be to use **Shiners** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Crankbaits**. The most successful lure is **Dim Green / Glow Green Crankbait 2.5 m, #3/0**.
