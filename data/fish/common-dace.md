# Common Dace

## How to catch

How to catch **a unique Common Dace**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Common Dace would be to use **Mayflies** on a size #2/0 hook.

If you are more into artificial lures, you should go for some **Nano Spinners**. The most successful lure is **Ladybug Spinner 4 g, #1/0**.
