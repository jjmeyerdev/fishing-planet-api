# Common Ling

## How to catch

How to catch **a unique Common Ling**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Common Ling would be to use **Giant Tiger Prawn** on a size #7/0 hook.

If you are more into artificial lures, you should go for some **Baby Halibuts**. The most successful lure is **Red/Yellow with glitter Baby Halibut 13 cm**.
