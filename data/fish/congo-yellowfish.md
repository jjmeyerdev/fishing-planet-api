# Congo Yellowfish

## How to catch

How to catch **a unique Congo Yellowfish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Congo Yellowfish would be to use **Night Crawlers** on a size #1 hook.

If you are more into artificial lures, you should go for some **Flat Spoons**. The most successful lure is **Yellow/Silver Trophy Flat Spoon 42 g, #6/0**.
