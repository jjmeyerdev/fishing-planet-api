# Cutthroat Trout

## How to catch

How to catch **a unique Cutthroat Trout**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Cutthroat Trout would be to use **Shiners** on a size #3/0 hook.

If you are more into artificial lures, you should go for some **Narrow Spoons**. The most successful lure is **Ammolite Barbless Narrow Spoon 7 g, #1/0**.
