# Dogtooth Tuna

## How to catch

How to catch **a unique Dogtooth Tuna**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Dogtooth Tuna would be to use **Japanese Flying Squid** on a size #10/0 hook.

If you are more into artificial lures, you should go for some **Pilkers**. The most successful lure is **Silver/Gold Trophy Double Hook Pilker 150 g, #7/0**.
