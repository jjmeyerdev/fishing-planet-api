# Gafftopsail Catfish

## How to catch

How to catch **a unique Gafftopsail Catfish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Gafftopsail Catfish would be to use **Shrimp** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Buzzbaits**. The most successful lure is **Acid Green and Orange Buzzbait 28 g, #5/0**.
