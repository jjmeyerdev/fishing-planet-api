# Grayling Wrestler

## How to catch

How to catch **a unique Grayling Wrestler**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Grayling Wrestler would be to use **Bloodworms** on a size #1 hook.

If you are more into artificial lures, you should go for some **Narrow Spoons**. The most successful lure is **Ammolite Narrow Spoon 7 g, #1/0**.
