# Great Barracuda

## How to catch

How to catch **a unique Great Barracuda**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Great Barracuda would be to use **Japanese Jack Mackerel** on a size #10/0 hook.

If you are more into artificial lures, you should go for some **Saltwater Poppers**. The most successful lure is **Orange/White Giant Popper 130 g, #7/0**.
