# Green Jobfish

## How to catch

How to catch **a unique Green Jobfish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Green Jobfish would be to use **Medium Fish Strips** on a size #6/0 hook.

If you are more into artificial lures, you should go for some **Pilkers**. The most successful lure is **Silver/Gold Trophy Double Hook Pilker 150 g, #7/0**.
