# Greenback Cutthroat Trout

## How to catch

How to catch **a unique Greenback Cutthroat Trout**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Greenback Cutthroat Trout would be to use **Shiners** on a size #3/0 hook.

If you are more into artificial lures, you should go for some **Single Spoons**. The most successful lure is **Yellow/Black Club-Series Single Spoon 42 g, #4/0**.
