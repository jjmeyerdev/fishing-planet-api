# Ide

## How to catch

How to catch **a unique Ide**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Ide would be to use **Small Minnows** on a size #2/0 hook.

If you are more into artificial lures, you should go for some **Crankbaits**. The most successful lure is **Perch Crankbait 2.5 m, #4/0**.
