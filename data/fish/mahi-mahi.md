# Mahi-Mahi

## How to catch

How to catch **a unique Mahi-Mahi**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Mahi-Mahi would be to use **Japanese Flying Squid** on a size #5/0 hook.

If you are more into artificial lures, you should go for some **Trolling Octopuses**. The most successful lure is **Black/Purple Trolling Octopus 85 g, #7/0**.
