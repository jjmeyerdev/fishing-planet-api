# Mega Piranha

## How to catch

How to catch **a unique Mega Piranha**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Mega Piranha would be to use **Blood** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Buzzbaits**. The most successful lure is **White Buzzbait 28 g, #5/0**.
