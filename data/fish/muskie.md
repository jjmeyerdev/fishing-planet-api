# Muskie

## How to catch

How to catch **a unique Muskie**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Muskie would be to use **Large Minnows** on a size #7/0 hook.

If you are more into artificial lures, you should go for some **Shads**. The most successful lure is **Pike Golem Lure 2 m, #8/0**.
