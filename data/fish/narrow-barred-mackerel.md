# Narrow-Barred Mackerel

## How to catch

How to catch **a unique Narrow-Barred Mackerel**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Narrow-Barred Mackerel would be to use **Japanese Jack Mackerel** on a size #10/0 hook.

If you are more into artificial lures, you should go for some **Bucktails**. The most successful lure is **Gold/Black/White Bucktail 50 g, #6/0**.
