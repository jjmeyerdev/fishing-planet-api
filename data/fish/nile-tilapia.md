# Nile Tilapia

## How to catch

How to catch **a unique Nile Tilapia**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Nile Tilapia would be to use **Bloodworms** on a size #1/0 hook.

If you are more into artificial lures, you should go for some **Slugs**. The most successful lure is **Yellow Grub 3 cm**.
