# Northern Pike

## How to catch

How to catch **a unique Northern Pike**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Northern Pike would be to use **Shiners** on a size #6/0 hook.

If you are more into artificial lures, you should go for some **Jerkbaits**. The most successful lure is **Pike Golem Lure 2 m, #8/0**.
