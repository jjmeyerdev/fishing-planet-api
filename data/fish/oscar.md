# Oscar

## How to catch

How to catch **a unique Oscar**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Oscar would be to use **Bloodworms** on a size #1/0 hook.

If you are more into artificial lures, you should go for some **Spinners**. The most successful lure is **Beetle Spinner 4 g, #1/0**.
