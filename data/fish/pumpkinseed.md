# Pumpkinseed

## How to catch

How to catch **a trophy Pumpkinseed**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Pumpkinseed would be to use **Red Worms** on a size #4 hook.

If you are more into artificial lures, you should go for some **Craws**. The most successful lure is **Green Pumpkin Craw 2.5 cm**.
