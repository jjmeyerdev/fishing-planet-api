# Rainbow Outlaw

## How to catch

How to catch **a unique Rainbow Outlaw**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Rainbow Outlaw would be to use **Brown Trout Eggs** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Spinners**. The most successful lure is **Yellow/Black X-Series Barbless Spinner 14 g, #3/0**.
