# Red Tail Barracuda

## How to catch

How to catch **a common Red Tail Barracuda**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Red Tail Barracuda would be to use **Centipedes** on a size #1 hook.

If you are more into artificial lures, you should go for some **Nano Spinners**. The most successful lure is **Yellow Tiger Bug Barbless Nano Spinner 4 g, #1**.
