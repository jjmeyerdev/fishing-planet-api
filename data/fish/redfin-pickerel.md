# Redfin Pickerel

## How to catch

How to catch **a trophy Redfin Pickerel**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Redfin Pickerel would be to use **Small Minnows** on a size #1/0 hook.

If you are more into artificial lures, you should go for some **Casting Spoons**. The most successful lure is **Yellow & Red Rombs Casting Spoon 7 g, #1/0**.
