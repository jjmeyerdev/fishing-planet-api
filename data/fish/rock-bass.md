# Rock Bass

## How to catch

How to catch **a common Rock Bass**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Rock Bass would be to use **Bloodworms** on a size #8 hook.

If you are more into artificial lures, you should go for some **Nano Spinners**. The most successful lure is **Nano Spoon 4 g, #1**.
