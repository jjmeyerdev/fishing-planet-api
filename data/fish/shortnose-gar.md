# Shortnose Gar

## How to catch

How to catch **a unique Shortnose Gar**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Shortnose Gar would be to use **Shiners** on a size #3/0 hook.

If you are more into artificial lures, you should go for some **Minnows**. The most successful lure is **Roach Hunched Runner 1 m, #1/0**.
