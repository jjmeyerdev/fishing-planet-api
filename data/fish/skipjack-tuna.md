# Skipjack Tuna

## How to catch

How to catch **a unique Skipjack Tuna**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Skipjack Tuna would be to use **Indian Ocean Squid** on a size #10/0 hook.

If you are more into artificial lures, you should go for some **Trolling Octopuses**. The most successful lure is **Black/Purple Trolling Octopus 85 g, #7/0**.
