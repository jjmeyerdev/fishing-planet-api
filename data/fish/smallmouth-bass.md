# Smallmouth Bass

## How to catch

How to catch **a unique Smallmouth Bass**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Smallmouth Bass would be to use **Crawfish** on a size #1/0 hook.

If you are more into artificial lures, you should go for some **Worms**. The most successful lure is **Yellow/Black X-Series Casting Spoon 21 g, #3/0**.
