# Southern Flounder

## How to catch

How to catch **a unique Southern Flounder**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Southern Flounder would be to use **Gulf Shrimp** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Buzzbaits**. The most successful lure is **Acid Green and Orange Buzzbait 28 g, #5/0**.
