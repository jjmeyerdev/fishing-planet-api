# Spotfin Flyingfish

## How to catch

How to catch **a trophy Spotfin Flyingfish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Spotfin Flyingfish would be to use **Shrimp** on a size #1 hook.

If you are more into artificial lures, you should go for some **Grubs**. The most successful lure is **Red Grub 5 cm**.
