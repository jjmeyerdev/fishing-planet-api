# Striped Bonito

## How to catch

How to catch **a unique Striped Bonito**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Striped Bonito would be to use **Shrimp** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Pilkers**. The most successful lure is **Silver/Gold Trophy Single Hook Pilker 60 g, #4/0**.
