# Swordfish

## How to catch

How to catch **a unique Swordfish**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Swordfish would be to use **Split Tail Mullet** on a size #12/0 hook.

If you are more into artificial lures, you should go for some **Trolling Octopuses**. The most successful lure is **Glowing Blue Glowing Trolling Octopus 230 g, #14/0**.
