# Walleye

## How to catch

How to catch **a unique Walleye**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Walleye would be to use **Leeches** on a size #4/0 hook.

If you are more into artificial lures, you should go for some **Worms**. The most successful lure is **Dim Green / Glow Green Worm 12.5 cm**.
