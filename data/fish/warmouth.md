# Warmouth

## How to catch

How to catch **a trophy Warmouth**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Warmouth would be to use **Bloodworms** on a size #4 hook.

If you are more into artificial lures, you should go for some **Craws**. The most successful lure is **Dark Space Blue Craw 2.5 cm**.
