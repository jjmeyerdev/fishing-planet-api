# Whiting

## How to catch

How to catch **a unique Whiting**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Whiting would be to use **Sea Worms** on a size #6/0 hook.

If you are more into artificial lures, you should go for some **Pilkers**. The most successful lure is **Silver/Yellow Sonar Fish Pilker 200 g, #4/0**.
