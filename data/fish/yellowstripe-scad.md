# Yellowstripe Scad

## How to catch

How to catch **a trophy Yellowstripe Scad**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Yellowstripe Scad would be to use **Sea Worms** on a size #1/0 hook.

If you are more into artificial lures, you should go for some **Grubs**. The most successful lure is **Red Grub 7 cm**.
