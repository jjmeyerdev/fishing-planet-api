# Zander

## How to catch

How to catch **a unique Zander**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Zander would be to use **Small Minnows** on a size #1 hook.

If you are more into artificial lures, you should go for some **Narrow Spoons**. The most successful lure is **Green Holographic Narrow Spoon 21 g, #3/0**.
