# Zombie Cat

## How to catch

How to catch **a unique Zombie Cat**? You can catch this fish either with **bait** or **lures**.

If you prefer to use natural baits, then the best way to catch Zombie Cat would be to use **Zombie Bait** on a size #6/0 hook.

If you are more into artificial lures, you should go for some **Spinnerbaits**. The most successful lure is **Lemon Skirt and Blades Spinnerbait 28 g, #4/0**.
